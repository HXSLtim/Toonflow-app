## 项目管理

### 创建项目

项目是 Toonflow 中的顶层组织单位，所有内容都归属于某个项目。

**项目属性**：
- **项目名称**：项目的标识名称
- **项目简介**：简短描述项目主题和内容
- **项目类型**：短剧、漫剧、MV 等
- **艺术风格**：写实、动漫、水墨、油画、赛博朋克等
- **视频比例**：
  - 16:9（横屏，适合电脑和电视）
  - 9:16（竖屏，适合手机短视频）
  - 1:1（方形，适合社交媒体）
  - 4:3（传统比例）

**使用场景**：
- 为不同的故事创建独立项目
- 同一故事的不同版本（如不同风格）可创建多个项目
- 测试和正式项目分开管理

### 项目设置

每个项目可以独立配置：
- AI 模型选择
- 默认艺术风格
- 视频参数
- 提示词模板

### 项目统计

查看项目数据：
- 章节数量
- 生成的剧本数
- 分镜总数
- 视频片段数
- 总时长
- API 调用次数和成本

## 小说管理

### 添加章节

支持多种方式添加小说内容：

**方式一：手动输入**
- 直接在编辑器中输入或粘贴文本
- 支持富文本格式
- 实时字数统计

**方式二：文件导入**
- 支持格式：TXT、DOCX、PDF
- 自动识别章节结构
- 批量导入多个文件

**方式三：在线抓取**（计划中）
- 从小说网站直接导入
- 自动分章节
- 保留原始格式

### 章节管理

**章节属性**：
- **章节名称**：如"第一章 相遇"
- **章节索引**：用于排序
- **所属卷**：可选，用于长篇小说分卷
- **章节内容**：小说正文
- **创建时间**：自动记录

**章节操作**：
- 编辑：修改章节内容
- 删除：移除章节（不可恢复）
- 排序：调整章节顺序
- 合并：将多个章节合并为一个
- 拆分：将长章节拆分为多个

### 文本预处理

系统会自动对文本进行预处理：
- 去除多余空格和换行
- 识别对话和旁白
- 提取角色名称
- 分析场景切换
- 标注情感基调

## 大纲生成

### AI 自动生成

基于小说内容，AI 会自动生成结构化大纲。

**生成流程**：
1. 选择要使用的章节
2. AI 分析文本内容
3. 识别关键情节点
4. 提取主要角色
5. 划分剧集结构
6. 生成每集大纲

**大纲包含**：
- 集数编号
- 集名称
- 主要场景
- 出场角色
- 核心剧情
- 情感基调
- 预计时长

### 手动编辑

可以对 AI 生成的大纲进行调整：
- 修改集数划分
- 调整剧情顺序
- 添加或删除场景
- 补充角色信息
- 标注重点镜头

### 故事线管理

**故事线功能**：
- 主线剧情：核心故事发展
- 支线剧情：次要情节线
- 角色弧光：角色成长轨迹
- 情感线：情感关系发展

**可视化展示**：
- 时间轴视图
- 关系图谱
- 情节树状图

## 剧本生成

### AI 剧本生成

基于大纲生成详细剧本。

**剧本格式**：
```
场景编号：场景名称 - 时间

[场景描述]

角色A：（情绪/动作）对白内容

[动作描述]

角色B：（情绪/动作）对白内容
```

**生成内容**：
- 场景描述：环境、氛围、道具
- 角色对白：符合角色性格的对话
- 动作指示：角色的动作和表情
- 情绪标注：角色的情感状态
- 镜头建议：推荐的拍摄角度

### 剧本编辑

**编辑功能**：
- 富文本编辑器
- 语法高亮
- 自动格式化
- 版本历史
- 协作批注（计划中）

**智能辅助**：
- 对白润色
- 场景扩写
- 角色一致性检查
- 时长预估

### 剧本导出

支持多种格式导出：
- PDF：标准剧本格式
- DOCX：可编辑文档
- Final Draft：专业编剧软件格式
- Fountain：纯文本剧本格式

## 分镜制作

### 自动分镜

AI 自动将剧本拆分为分镜。

**分镜要素**：
- **镜头编号**：如 S01-001
- **场景**：拍摄地点和环境
- **角色**：出现的角色及位置
- **镜头类型**：
  - 特写（Close-up）
  - 中景（Medium shot）
  - 全景（Full shot）
  - 远景（Long shot）
  - 过肩镜头（Over-the-shoulder）
- **机位**：摄像机角度
  - 平视
  - 俯视
  - 仰视
  - 侧面
- **运镜**：
  - 固定镜头
  - 推镜头
  - 拉镜头
  - 摇镜头
  - 跟镜头
- **时长**：镜头持续时间
- **对白**：该镜头的台词
- **音效**：背景音和音效

### 图片生成提示词

为每个分镜生成图片提示词。

**提示词结构**：
```
[主体描述], [环境描述], [光线描述], [色彩描述],
[构图描述], [艺术风格], [质量标签]
```

**示例**：
```
A young woman with long black hair, wearing a white dress,
standing in a modern coffee shop, warm afternoon sunlight
streaming through large windows, soft bokeh background,
cinematic composition, realistic style, highly detailed,
8k resolution, professional photography
```

### 提示词优化

**润色功能**：

**优化选项**：

### 图片生成

### 分镜编辑

## 视频合成

### 视频配置

**基础配置**：
- **视频模型**：选择视频生成服务
- **生成模式**：
  - 文生视频（Text-to-Video）
  - 图生视频（Image-to-Video）
  - 视频续写（Video-to-Video）
- **视频时长**：5秒、10秒、15秒等
- **分辨率**：
  - 720p (1280x720)
  - 1080p (1920x1080)
  - 2K (2560x1440)
  - 4K (3840x2160)
- **帧率**：24fps、30fps、60fps

**高级配置**：
- 运动强度
- 镜头稳定性
- 色彩风格
- 音频选项

### 视频生成

**生成流程**：
1. 选择分镜图片
2. 配置视频参数
3. 生成视频提示词
4. 提交生成任务
5. 等待生成完成
6. 预览和下载

**批量生成**：
- 选择多个分镜
- 使用相同配置
- 队列管理
- 进度监控

### 视频编辑

**在线编辑器**（集成）：
- 视频裁剪
- 添加转场
- 插入字幕
- 背景音乐
- 音效同步
- 色彩校正
- 特效添加

**导出选项**：
- 视频格式：MP4、MOV、AVI
- 编码器：H.264、H.265
- 比特率：自定义
- 水印：可选

- Anthropic (Claude 3.5, Claude 3)
- DeepSeek (DeepSeek-V3)
- 阿里云（通义千问）
- 智谱 AI (GLM-4)

- 讯飞（星火）
- 月之暗面（Kimi）
- 其他兼容 OpenAI API 的服务

**配置项**：
- API Key
- Base URL（可选）
- 模型名称
- 温度参数
- 最大 Token 数
- 超时设置

### 图片生成模型

**支持的服务商**：
- OpenAI (DALL-E 3)
- Stability AI (Stable Diffusion)
- Midjourney (API)
- Nano Banana Pro
- 文心一格
- 通义万相
- 自部署模型（ComfyUI、Automatic1111）

**配置项**：
- API 地址
- 认证信息
- 默认参数
- 图片尺寸
- 生成数量

### 视频生成模型

**支持的服务商**：
- OpenAI Sora
- 豆包视频
- Runway Gen-3
- Pika Labs
- Stable Video Diffusion

**配置项**：
- API 配置
- 视频参数
- 质量设置
- 并发限制

### 模型映射

为不同任务分配合适的模型：

| 任务类型 | 推荐模型 | 说明 |
|---------|---------|------|
| 大纲生成 | GPT-3.5 / DeepSeek | 成本低，效果好 |
| 剧本生成 | GPT-4 / Claude | 质量高，逻辑强 |
| 分镜生成 | GPT-4 | 需要理解视觉构图 |
| 提示词优化 | GPT-3.5 | 快速响应 |
| 图片生成 | DALL-E 3 / SD | 根据风格选择 |
| 视频生成 | Sora / 豆包 | 根据预算选择 |
