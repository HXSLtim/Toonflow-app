## 安装与启动

### 方式一：下载安装包（推荐）

1. 访问 [GitHub Releases](https://github.com/HBAI-Ltd/Toonflow-app/releases) 或 [夸克网盘](https://pan.quark.cn/s/94ef07509df0)
2. 下载适合你操作系统的安装包
3. 运行安装程序
4. 启动 Toonflow

### 方式二：Docker 部署

```bash
# 快速启动（使用 GitHub 源）
docker-compose -f docker/docker-compose.yml up -d --build

# 国内用户推荐使用 Gitee 源
GIT=gitee docker-compose -f docker/docker-compose.yml up -d --build
```

启动后访问：`http://localhost:随机端口`（查看终端输出获取端口号）

### 方式三：云端部署

```bash
# 安装 Node.js 24.x
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.7/install.sh | bash
source ~/.bashrc
nvm install 24

# 克隆项目
git clone https://github.com/HBAI-Ltd/Toonflow-app.git
cd Toonflow-app

# 安装依赖并构建
yarn install
yarn build

# 启动服务
pm2 start pm2.json
```

## 首次登录

1. 打开浏览器访问 Toonflow（本地安装会自动打开）
2. 使用默认账号登录：
   - 账号：`admin`
   - 密码：`admin123`
3. 输入验证码
4. 点击登录

**安全提示**：首次登录后，请立即修改默认密码！

## 创建第一个项目

### 步骤 1：新建项目

1. 点击左侧菜单「项目管理」
2. 点击「新建项目」按钮
3. 填写项目信息：
   - **项目名称**：例如「我的第一个短剧」
   - **项目简介**：简单描述你的故事主题
   - **项目类型**：选择「短剧」
   - **艺术风格**：选择「写实」、「动漫」或「水墨」等
   - **视频比例**：选择「16:9」（横屏）或「9:16」（竖屏）
4. 点击「创建」

### 步骤 2：上传小说文本

1. 进入刚创建的项目
2. 点击「小说管理」
3. 点击「添加章节」
4. 填写章节信息：
   - **章节名称**：例如「第一章 相遇」
   - **章节内容**：粘贴或输入你的小说文本
   - **章节索引**：1
5. 点击「保存」

**提示**：可以一次性上传多个章节，系统会自动识别章节结构。

## 配置 AI 模型

在开始创作前，需要配置 AI 服务。

### 步骤 1：进入设置页面

1. 点击右上角「设置」图标
2. 选择「AI 模型配置」

### 步骤 2：添加大语言模型（必需）

1. 点击「添加模型」
2. 选择厂商：
   - OpenAI (GPT-4, GPT-3.5)
   - Anthropic (Claude)
   - DeepSeek
   - 通义千问
   - 智谱 AI
   - 其他兼容 OpenAI 格式的服务
3. 填写配置：
   - **API Key**：你的 API 密钥
   - **Base URL**：API 地址（可选，使用默认值）
   - **模型名称**：例如 `gpt-4`
4. 点击「测试连接」确认配置正确
5. 点击「保存」

### 步骤 3：添加图片生成模型（必需）

1. 点击「添加图片模型」
2. 选择厂商：
   - DALL-E 3
   - Stable Diffusion
   - Midjourney
   - Nano Banana Pro（推荐）
3. 填写 API 配置
4. 测试并保存

### 步骤 4：添加视频生成模型（必需）

1. 点击「添加视频模型」
2. 选择厂商：
   - Sora
   - 豆包视频
   - Runway
3. 填写 API 配置
4. 测试并保存

### 步骤 5：配置模型映射

为不同的任务分配合适的模型：

- **剧本生成**：选择一个强大的大语言模型（推荐 GPT-4 或 Claude）
- **分镜生成**：选择大语言模型
- **图片生成**：选择图片模型
- **视频生成**：选择视频模型

## 完整创作流程

### 第一步：生成大纲

1. 在项目中点击「大纲管理」
2. 点击「AI 生成大纲」
3. 选择要使用的小说章节
4. 点击「开始生成」
5. 等待 AI 分析并生成故事大纲
6. 查看生成的大纲，可以手动编辑调整

**大纲示例**：
```
第一集：相遇
- 场景：都市咖啡厅
- 主要角色：李明（男主）、王小雅（女主）
- 剧情：李明在咖啡厅偶遇王小雅，两人因一杯咖啡结缘
- 情感基调：轻松、浪漫
```

### 第二步：生成剧本

1. 选择一个大纲
2. 点击「生成剧本」
3. AI 会根据大纲生成详细的剧本，包括：
   - 场景描述
   - 角色对白
   - 动作指示
   - 情绪标注
4. 查看并编辑剧本

**剧本示例**：
```
场景1：咖啡厅内 - 下午

[李明推门而入，环顾四周]

李明：（自言自语）今天人真多啊...

[王小雅正在靠窗的位置看书，阳光洒在她身上]

李明：（走向吧台）一杯美式，谢谢。

[服务员递上咖啡，李明转身时不小心撞到了王小雅]

李明：（慌张）对不起对不起！

王小雅：（抬头，微笑）没关系。
```

### 第三步：生成分镜

1. 进入「分镜管理」
2. 选择剧本
3. 点击「生成分镜」
4. AI 会自动将剧本拆分为多个镜头，每个镜头包括：
   - 镜头编号
   - 场景描述
   - 角色位置
   - 镜头类型（特写、中景、远景等）
   - 图片生成提示词
5. 查看分镜列表

### 第四步：生成分镜图片

1. 在分镜列表中选择一个镜头
2. 点击「生成图片」
3. AI 会根据提示词生成图片
4. 如果不满意，可以：
   - 点击「重新生成」
   - 编辑提示词后再生成
   - 使用「润色提示词」功能优化
   - 手动上传自己的图片

**提示词优化技巧**：
- 使用「润色提示词」功能，AI 会自动优化提示词
- 添加更多细节描述（光线、色彩、构图等）
- 指定艺术风格（写实、动漫、油画等）

### 第五步：批量超分（可选）

如果生成的图片分辨率不够：

1. 选择多个分镜
2. 点击「批量超分」
3. 系统会自动提升图片分辨率和质量

### 第六步：生成视频提示词

1. 为每个分镜生成视频提示词
2. 点击「生成视频提示词」
3. AI 会根据图片和剧本生成适合视频生成的提示词
4. 可以手动调整提示词

### 第七步：生成视频

1. 进入「视频管理」
2. 点击「新建视频配置」
3. 选择：
   - 视频模型
   - 生成模式（文生视频 / 图生视频）
   - 视频时长（5秒 / 10秒等）
   - 分辨率
4. 选择要使用的分镜图片
5. 点击「生成视频」
6. 等待视频生成完成

### 第八步：导出与编辑

1. 视频生成完成后，可以：
   - 下载单个视频片段
   - 批量导出所有视频
   - 使用在线编辑器进行后期处理
2. 添加背景音乐、字幕、转场效果
3. 导出最终成品

## 实用技巧

### 提高生成质量

1. **小说文本准备**
   - 确保文本格式清晰，段落分明
   - 包含足够的场景和对话描述
   - 避免过于简短或过于冗长的章节

2. **大纲优化**
   - 生成后仔细检查大纲逻辑
   - 手动调整不合理的情节
   - 确保每集时长适中（建议 3-5 分钟）

3. **剧本调整**
   - 检查对话是否自然
   - 补充必要的场景描述
   - 标注重要的情感和动作

4. **分镜优化**
   - 使用「润色提示词」功能
   - 添加具体的视觉细节
   - 保持角色外观一致性

5. **视频生成**
   - 选择合适的视频时长（不要太长）
   - 使用高质量的分镜图片
   - 测试不同的提示词风格

### 节省成本

1. **模型选择**
   - 大纲和剧本生成：使用 GPT-3.5 或 DeepSeek（成本低）
   - 重要场景：使用 GPT-4 或 Claude（质量高）
   - 图片生成：批量生成可降低单价

2. **批量处理**
   - 一次性生成多个分镜
   - 使用批量超分功能
   - 批量生成视频

3. **复用素材**
   - 保存常用的角色设定
   - 复用相似场景的图片
   - 建立自己的素材库

### 提高效率

1. **使用模板**
   - 保存常用的提示词模板
   - 创建项目模板
   - 复用成功的配置

2. **并行处理**
   - 同时处理多个项目
   - 在等待 AI 生成时准备下一步内容
   - 使用任务队列管理

3. **快捷操作**
   - 使用键盘快捷键
   - 批量选择和操作
   - 收藏常用功能

## 常见问题

### Q1: 登录后无法访问功能？

**A**: 检查以下几点：
- Token 是否过期（24小时有效期）
- 浏览器是否禁用了 Cookie
- 尝试清除浏览器缓存后重新登录

### Q2: AI 生成速度很慢？

**A**: 可能的原因：
- API 服务响应慢（检查网络连接）
- 模型负载高（尝试换个时间段）
- 提示词过于复杂（简化提示词）

### Q3: 生成的内容不符合预期？

**A**: 优化建议：
- 使用「润色提示词」功能
- 手动编辑提示词，添加更多细节
- 尝试重新生成（每次结果可能不同）
- 更换更强大的 AI 模型

### Q4: 图片生成失败？

**A**: 检查：
- API Key 是否正确
- 账户余额是否充足
- 提示词是否包含敏感内容
- 查看错误日志获取详细信息

### Q5: 视频生成失败？

**A**: 常见原因：
- 图片分辨率不符合要求（使用超分功能）
- 视频时长设置不合理
- API 配额不足
- 提示词不符合视频模型要求

### Q6: 如何保持角色外观一致？

**A**: 建议：
- 在「素材管理」中保存角色参考图
- 在提示词中详细描述角色特征
- 使用相同的艺术风格设置
- 考虑使用角色 LoRA 模型（高级功能）

### Q7: 可以导入已有的剧本吗？

**A**: 可以！
- 在「剧本管理」中选择「导入剧本」
- 支持 TXT、DOCX 等格式
- 系统会自动解析剧本结构

### Q8: 如何备份我的项目？

**A**:
- 使用「导出项目」功能
- 定期备份数据库文件
- 导出所有生成的素材

### Q9: 支持多人协作吗？

**A**: 当前版本暂不支持多人协作，但可以：
- 导出项目文件分享
- 使用云端部署实现团队访问
- 期待后续版本的协作功能
